neo4j
=====

Neo4j client for [Go](http://golang.org).


# Documentation

See [GoDoc](http://godoc.org/github.com/jmcvetta/neo4j) for automatic
documentation.


# Status

This driver is a work in progress.  It is currently incomplete, and not ready
for production use.
